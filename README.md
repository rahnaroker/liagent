# litagent

**Редактор и конвертер электронных книг: FB2 → EPUB3 для Kindle.**

Портативное офлайн-приложение на [Wails](https://wails.io) (ядро на Go + интерфейс на Svelte).

---

## Зачем это нужно

FB2-файлы (особенно полученные из сканов и OCR) часто содержат типографский «мусор»:
двойные пробелы, дефисы вместо тире, прямые кавычки, разорванные переносами слова,
битую кодировку windows-1251. **litagent** приводит текст в порядок и собирает из него
чистый EPUB3, готовый для отправки в Kindle (*Send to Kindle* принимает EPUB).

Что он делает:

- **Чинит типографику** по набору правил — пробелы, тире, кавычки-«ёлочки», многоточия,
  неразрывные пробелы (предлоги, инициалы, числа), склейку переносов, OCR-артефакты.
- **Конвертирует в качественный EPUB3** — корректная разбивка на главы, рабочая навигация
  (nav + ncx), всплывающие сноски на Kindle, аккуратный Kindle-safe CSS, обложка.
- **Показывает все правки** сгруппированными по правилам — ничего не применяется «вслепую»,
  каждое изменение можно отключить.
- **Генерирует обложки** из шаблонов, если в книге её нет.
- **Редактирует метаданные** — название, авторы, переводчики, серия, издатель, аннотация.

Правки разделены по уровню уверенности: **A** (безопасные) и **B** (эвристические)
применяются автоматически, **C** — только подсказки для ручного ревью. Подробности
логики — в [`DESIGN.md`](DESIGN.md).

## Сборка

Требуется: **Go 1.25+**, **Node.js 18+**, **Wails CLI v2**.
WebView2 на Windows 11 уже встроен; на «чистых» системах его ставит установщик Edge/WebView2 Runtime.

```bash
# 1. установить Wails CLI (один раз)
go install github.com/wailsapp/wails/v2/cmd/wails@v2.12.0

# 2. собрать приложение
wails build
```

Готовый бинарник появится в `build/bin/litagent.exe`. Зависимости фронтенда (`npm install`)
и сборку Svelte Wails выполняет сам.

**Разработка** с горячей перезагрузкой интерфейса:

```bash
wails dev
```

**CLI-режим** (конвертация без GUI, удобно для пакетной обработки):

```bash
go build -o litagent-cli ./cmd/cli
litagent-cli книга.fb2 [книга.epub]
```

## Как пользоваться

1. Запустите `litagent.exe` и нажмите **«Открыть FB2…»**.
2. Книга разбирается, под шапкой появляется сводка (название, автор, язык, число глав,
   количество найденных правок). Дальше работайте по вкладкам (см. таблицу ниже).
3. Когда результат устраивает — нажмите **«Собрать EPUB…»** и выберите, куда сохранить.

| Вкладка | Что делает |
|---|---|
| **Правки** | Список всех найденных правок, сгруппированный по правилам, с диффом «было → стало». Слева — правила со счётчиками и галочками: снимите галку у правила **A/B**, чтобы откатить его, или поставьте у подсказки **C**, чтобы применить. Клик по названию правила фильтрует список. |
| **Превью** | Предпросмотр готовой книги в том виде, в каком она попадёт в EPUB. |
| **Обложка** | Генерация обложки из шаблонов (см. ниже). |
| **Метаданные** | Редактирование названия, авторов, переводчиков, языка, издателя, серии, даты, ISBN, ключевых слов и аннотации. Изменения применяются при сборке. Кнопка «убрать теги» чистит `[мусор]` из названия. |

> Можно открыть другую книгу в любой момент — активная вкладка сразу перезагрузится под неё.

### Обложки

Рядом с `litagent.exe` лежит папка **`wallpaper`** — это шаблоны фонов. В комплекте
идёт 11 готовых шаблонов: при сборке они автоматически копируются рядом с бинарником,
так что генерация обложек работает сразу. Список меняется свободно: просто положите или
удалите картинки (`.jpg`, `.png`, `.gif`, `.webp`).

На вкладке **«Обложка»**:

- **Сгенерировать** — берёт случайный фон и рисует по центру верхней половины белую плашку
  с названием книги и автором (серифный шрифт Georgia). Сразу помечает обложку к сборке.
- **Другой шаблон** — выбирает другой случайный фон.
- **Использовать в EPUB** — галочка, подставлять ли сгенерированную обложку при сборке.

По умолчанию обложка сохраняет **размер и пропорции** исходного шаблона — подбирайте
картинки под экран вашего Kindle (например 600×800), чтобы не было чёрных полос.

Тонкая настройка (необязательно) — файл `wallpaper/cover.json`. Поля `width`/`height` можно
не указывать (берётся размер шаблона) либо задать, чтобы привести все обложки к одному размеру:

```json
{ "jpegQuality": 88,
  "plateWidth": 0.80, "plateCenterY": 0.30,
  "fontTitle": "C:\\Windows\\Fonts\\georgiab.ttf",
  "fontAuthor": "C:\\Windows\\Fonts\\georgia.ttf" }
```

Папку шаблонов можно переопределить переменной окружения `LITAGENT_WALLPAPER`.
